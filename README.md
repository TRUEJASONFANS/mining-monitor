# 矿产储量监控工具

实时监控上市公司采矿权、矿产储量、探矿权相关公告

## 功能

- 监控关键词：采矿权、矿产储量、探矿权、矿业权
- 数据源：巨潮资讯网（证监会指定上市公司公告平台）
- 自动推送系统托盘通知
- 去重过滤，不重复提醒

## 安装

```bash
pip install requests plyer
```

## 运行

```bash
python mining_monitor.py
```

## 打包 EXE

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name="矿产监控" mining_monitor.py
```

## GitHub Actions 自动构建

每次 push 到 main 分支会自动构建 Windows EXE，可在 Actions 页面下载。

发布版本时打标签会自动发布到 Release：
```bash
git tag v1.0
git push origin v1.0
```
